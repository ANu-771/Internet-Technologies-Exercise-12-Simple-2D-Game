# 🚗 Simple 2D Car Game

A browser-based **2D Car Racing Game** developed using **HTML, CSS, JavaScript, and jQuery** as part of the Internet Technologies coursework assignment.

This project demonstrates DOM manipulation, game logic implementation, animations, collision handling, score calculation, and Object-Oriented Programming (OOP) concepts without using external game engines or frameworks.

---

## 📌 Project Overview

The objective of this project is to develop a simple single-player 2D game using frontend web technologies.

The game is inspired by classic traffic racing games where the player controls a car, avoids obstacles, progresses through multiple levels, and earns scores based on gameplay performance.

---

## 🎯 Assignment Objectives

- Develop a simple 2D browser game
- Implement game logic using JavaScript
- Practice DOM manipulation techniques
- Apply Object-Oriented Programming concepts
- Create level progression system
- Calculate and display player scores
- Build animations using CSS and JavaScript
